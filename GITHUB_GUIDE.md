# 🍎 깃허브(GitHub)로 무료 배포하는 방법

네, 가능합니다! **깃허브 페이지(GitHub Pages)** 기능을 쓰면 **평생 무료**로 게임을 배포할 수 있습니다.
코딩을 몰라도 파일을 **"마우스로 끌어서 업로드"**하면 끝납니다. 천천히 따라해보세요!

---

## 1단계: 저장소(Repository) 만들기
1.  **[GitHub.com](https://github.com/)**에 로그인합니다. (계정이 없다면 'Sign up'으로 가입해주세요.)
2.  오른쪽 위의 **`+`** 버튼을 누르고 **`New repository`**를 클릭합니다.
3.  **Repository name**에 원하는 영문 이름(예: `my-fruit-game`)을 입력합니다.
4.  나머지는 건드리지 말고 맨 아래 **`Create repository`** 초록색 버튼을 누릅니다.

## 2단계: 파일 업로드하기
1.  새로 만든 저장소 화면에서 **`uploading an existing file`**이라는 파란색 글씨를 찾아서 클릭합니다.
    *   (못 찾겠으면 주소창 뒤에 `/upload`를 붙여도 됩니다. 예: `github.com/내아이디/my-fruit-game/upload`)
2.  내 컴퓨터의 **`C:\Users\ghost\Downloads\게임_테스트\falling-fruit`** 폴더를 엽니다.
3.  이 폴더 안에 있는 **3개 파일**을 모두 선택해서 깃허브 화면에 끌어다 놓습니다.
    *   `index.html`
    *   `style.css`
    *   `game.js`
    *   (나머지 파일들은 안 올려도 됩니다)
4.  파일이 다 올라가면 아래쪽 **`Commit changes`** 버튼을 누릅니다.

## 3단계: 배포 설정 켜기 (가장 중요!)
1.  저장소 상단 메뉴 중 **`Settings` (설정)** 탭을 클릭합니다.
2.  왼쪽 메뉴에서 **`Pages`**를 찾아서 클릭합니다. (밑쪽에 있습니다)
3.  **"Build and deployment"** 섹션 아래 **Branch** 부분을 찾습니다.
    *   `None` 이라고 되어있는 것을 👉 **`main`** 으로 바꿉니다.
    *   옆에 폴더는 `/(root)` 그대로 둡니다.
    *   **`Save`** 버튼을 누릅니다.

## 4단계: 주소 확인하기
1.  이제 1~3분 정도 기다립니다. (깃허브가 열심히 사이트를 만들고 있습니다 🔨)
2.  다시 **`Settings` -> `Pages`** 메뉴로 들어가 봅니다.
3.  맨 위에 **"Your site is live at..."** 라는 메시지와 함께 **인터넷 주소**가 생깁니다!
4.  그 주소를 클릭하면 게임이 실행됩니다. 이제 이 주소를 친구들에게 보내주세요! 🎉
